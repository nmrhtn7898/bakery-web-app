# Bread Project
##### 애플리케이션 실행(로컬 환경)
- 애플리케이션 실행에 필요한 Redis, Mysql, Nginx 도커 컨테이너로 실행하기 위해 
docker-compose 명령어 실행
- docker-compose -f docker/default/docker-compose.yml up -d
- 컨테이너 실행 후, 애플리케이션 실행 및 디버깅
